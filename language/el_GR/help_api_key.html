1. Πηγαίνετε στη σελίδα του Instagram και συνδεθείτε με το λογαριασμό σας
  2. Πηγαίνετε στη σελίδα του Developper κάντε κλικ στο Register Your Application
  3. Θα σας ζητηθεί το url της ιστοσελίδα σας, ο αριθμός τηλεφώνου σας (μπορείτε να δώσετε ένα ψεύτικο αριθμό) και μια σύντομη περιγραφή
  4. Υποβολή και κάντε κλικ ξανά στο Register Your Application
  5. Εισάγετε ένα όνομα (το όνομα δεν μπορεί να περιέχει τη λέξη Instagram), μια περιγραφή και διεύθυνση της ιστοσελίδας σας πάλι
  6. στο πεδίο OAuth redirect_uri, κάντε επικόλληση το abobe URL και κάντε κλικ στο Register
  7. Τώρα κάντε αντιγραφή το Client ID και το Client Secret στη διαμόρφωση του Instagram2Piwigo
  8. Τελειώσατε!
