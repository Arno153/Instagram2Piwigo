1. Gå til Instagrams hjemmeside og log på
  2. Gå til Developper-siden og klik på Register Your Application
  3. Du vil blive bedt om din hjemmesides URL, dit telefonnummer (du kan oplyse et ugyldigt) og en kort beskrivelse
  4. Indsend og klik igen på Register Your Application
  5. Skriv dit navn (navnet må ikke indeholde ordet instagram), en beskrivelse og din hjemmesides URL igen
  6. Kopier ovennævnte URL til feltet OAuth redirect_uri og klik på Register
  7. Kopier dernæst Client ID og Client Secret til Instagram2Piwigos opsætning
  8. Sådan!
