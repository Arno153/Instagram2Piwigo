1. Ísť na Instagram webstránku a konektnúť sa s Vašimi loginmi
  2. Ísť na Developper stránku a kliknúť na Register Your Application
  3. Budete dotazovaný na url Vašej stránky, Vaše telefónne číslo (môžete zadať nesprávne číslo) a krátky popis
  4. Pošlite a kliknite opäť na Register Your Application
  5. Zadajte meno (meno nemôže obsahovať slovo instagram), popis a url Vašej web stránky opäť
  6. Do kolónky OAuth redirect_uri, prekopírujte URL a kliknite na Register
  7. Teraz prekopírujte Client ID a Client Secret do Instagram2Piwigo configuration
  8. Hotovo!
