1. Gå til Instagram nettside og logg inn
  2. Gå til Developer siden og klikk på Register Your Application
  3. Du vil bli bedt om din webadresse, ditt telefon nummer (du kan oppgi et falsk nummer) og en kort beskrivelse
  4. Send inn og klikk igjen på Register Your Application
  5. Skriv inn et navn (navnet kan ikke inneholde ordet instagram), en beskrivelse og din webadresse igjen
  6. På feltet OAuth redirect_uri, lim inn webadressen fra ovenfor og klikk på Registrer
  7. Nå kopier Client ID og Client Secret inn i Instagram2Piwigo konfigurasjon
  8. Du er ferdig!
