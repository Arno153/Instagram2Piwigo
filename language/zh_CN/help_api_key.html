1. 到 Instagram 网站然后用你的帐号登录
  2. 到 Developper 页面点击 Register Your Application
  3. 你将会被提示输出你的网站地址，你的电话号码（可填写个假的号码）和简短的说明
  4. 提交然后再次点击 Register Your Application
  5. 输入一个名称（此名称不能包含 instagram这个单词), 一个描述和你的网站地址
  6. 在 OAuth redirect_uri 里粘贴以上点击Register的URL
  7. 现在拷贝 Client ID 和 Client Secret 到 Instagram2Piwigo 配置文件里
  8. 你已完成啦！
