1. Пройдите по ссылке Instagram и авторизуйтесь
  2. Пройдите на страницу Developer и кликните Register Your Application
  3. Введите URL сайта, Ваш номер телефона (можно дать несуществующий номер) и краткое описание
  4. Сохраните данные и снова кликните на Register Your Application
  5. Введите имя (не может содержать слово instagram), описание и снова URL сайта
  6. Вставьте URL, указанный выше, в поле OAuth redirect_uri и кликните по кнопке Register
  7. Теперь введите полученные Client ID и Client Secret на странице настроек Instagram2Piwigo
  8. Готово!
