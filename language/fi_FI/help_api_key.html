1. Mene Instagram sivulle ja yhdistä kirjautumistunnuksesi
  2. Mene Developer sivulle ja klikkaa Register Your Application
  3. Sinua pyydetään lisäämään sivustosi osoite, puhelinnumerosi (voit antaa tekaistun numeron) ja lyhyt kuvaus
  4. Lähetä tiedot ja paina uudelleenRegister Your Application
  5. Lisää sivustolle nimi (ei saa sisältää sanaa instagram), kuvaus ja sivustosi osoite uudelleen
  6. Kopioi ja liitä kenttään OAuth redirect_uri URL-osoite yltä ja paina Register
  7. Nyt voit kopioida Client ID ja Client Secret tunnisteet Instagram2Piwigo asetuksiin
  8. Valmista tuli!
