1. Gå till Instagram och anslut med din inloggning
  2. Gå till sidan Developer och klicka påRegister Your Application
  3. Du blir ombedd att lämna din webbsidas url, ditt telefonnummer (behöver inte vara riktigt) och en kort beskrivning
  4. Fyll i och klicka på Register Your Application
  5. Ange ett namn som inte innehåller ordet instagram, en beskrivning och url till din webbsida på nytt
  6. Klistra in URL:en ovan i fältet OAuth redirect_uri och klicka på Register
  7. Kopiera Client ID och Client Secret till Instagram2Piwigos konfiguration
  8. Klart!
