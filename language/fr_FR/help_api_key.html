1. Aller sur le site Instagram et connectez-vous à votre compte
  2. Allez sur la page Developper et cliquez sur Register Your Application
  3. Vous devrez renseigner l'adresse de votre site, votre numéro de téléphone (vous pouvez mettre un faux) et une courte description
  4. Validez et cliquez à nouveau sur Register Your Application
  5. Entrez un nom (le nom ne peut pas contenir le mot instagram), une description et à nouveau l'adresse de votre site
  6. Dans le champs OAuth redirect_uri, copiez-collez l'URL ci-dessus et cliquez sur Register
  7. Maintenant copiez le Client ID et le Client Secret dans la configuration d'Instagram2Piwigo
  8. Terminé!
